# Python-кейс: кластеризация клиентов банка

## Что внутри

- `data/bank_client_behavior.csv` - синтетический датасет по клиентам банка;
- `data/cluster_profiles.csv` - агрегированные профили кластеров;
- `data/target_clients_for_campaign.csv` - готовый shortlist для коммуникации;
- `data/target_clients_from_notebook.csv` - файл, который пересохраняется из тетради;
- `client_clustering_walkthrough.ipynb` - пошаговая тетрадь с текстами и комментариями на русском;
- `artifacts/cluster_report.html` - готовый локальный HTML-отчет;
- `artifacts/cluster_scatter_2d.png` и `artifacts/cluster_scatter_3d.png` - готовые графики для слайдов;
- `artifacts/cluster_profiles.png` - сравнение профилей кластеров;
- `open_notebook.sh` - быстрый запуск тетради;
- `run_cluster_case.sh` - открытие готового HTML-отчета.

## Зачем здесь именно кластеризация

Простая сегментация по диапазонам не решает задачу. В данных есть клиенты с высоким оборотом, хорошей платежной дисциплиной и отсутствием просрочек, но при этом с низкой маржой для банка. Такой профиль проявляется только на комбинации нескольких признаков, поэтому в кейсе используется `K-means`.

## Что нужно для показа в IDE

- установленный `Python 3.10+`;
- `VS Code` или другая IDE с поддержкой Jupyter notebooks;
- для VS Code: расширения `Python` и `Jupyter`.

## Как открыть и показать тетрадь в VS Code

1. Установите `Python` и `VS Code`.
2. В VS Code поставьте расширения `Python` и `Jupyter`.
3. Откройте папку `python_client_clusters`.
4. Откройте файл `client_clustering_walkthrough.ipynb`.
5. Выберите интерпретатор Python или kernel.
6. Запускайте ячейки по шагам кнопкой `Run Cell` или сразу `Run All`.

## Что происходит в тетради

1. Загрузка CSV.
2. Отбор признаков для кластеризации.
3. Нормализация данных.
4. Подбор количества кластеров через `silhouette score` и `inertia`.
5. Обучение `K-means`.
6. Визуализация кластеров в 2D и 3D через `PCA`.
7. Построение профиля кластеров.
8. Выбор целевого кластера и выгрузка списка клиентов для контакта.

## Быстрый запуск

- `./open_notebook.sh` - открыть тетрадь через `jupyter lab`, если он установлен.
- `./run_cluster_case.sh` - открыть готовый HTML-отчет в браузере.

## Что удобно проговорить на занятии

- Простая сегментация по одному признаку не показывает скрытый убыточный паттерн.
- После кластеризации видно, какие клиенты выглядят “хорошими” по базовым метрикам, но мало зарабатывают банку.
- Python удобен, когда нужно не только считать метрики, но и применять модель, визуализировать результат и выгружать список клиентов для действий.
