# Excel-кейс: пилот ИИ для юристов в банке

Практический учебный кейс для занятия по импорту, очистке, агрегации и визуализации данных в Excel.

## Что внутри

- `data/legal_ai_contract_review_raw.csv` - большая сырая выгрузка для импорта через `Данные -> Из текста/CSV`.
- `data/legal_ai_contract_review_clean.csv` - очищенный вариант для сравнения результата.
- `data/legal_ai_contract_review_training.xlsx` - учебная книга со всеми листами, формулами и графиками.
- `artifacts/excel_before_after.png` - иллюстрация для слайда про очистку.
- `artifacts/excel_pivot_placeholder.png` - иллюстрация для слайда про сводную таблицу.
- `artifacts/excel_final_decision.png` - финальный график для управленческого решения.
- `artifacts/data_category_map_legal_ai.svg` - схема получения данных.

## Что происходит в кейсе

Банк тестирует ИИ-помощника для юристов и хочет понять, сокращается ли время согласования договоров. Сырые логи содержат несколько типовых проблем:

- сотрудник открыл новый документ, но предыдущее событие закрытия не пришло;
- документ остался открытым на ночь или на выходные, и длительность искусственно выросла;
- часть полей пришла пустой или с шумом.

В workbook это разбирается по шагам:

1. Импорт сырого CSV в Excel.
2. Восстановление закрытий через следующий `open` того же сотрудника.
3. Исключение нерабочего времени из длительности.
4. Сравнение метрики до и после очистки.
5. Нативная сводная таблица по месяцам и типам договоров с цветовой подсветкой.
6. Финальная визуализация с доверительным интервалом, где расчет идет формулами от очищенных данных.

## Как показывать на занятии

1. В Excel откройте `Данные -> Из текста/CSV` и импортируйте `legal_ai_contract_review_raw.csv`.
2. Покажите, что это большая выгрузка для реального сценария обучения.
3. Откройте `legal_ai_contract_review_training.xlsx`.
4. На листе `02_Cleaning` покажите служебные столбцы `next_open_same_employee`, `corrected_close`, `clean_work_minutes`.
5. На листе `03_Before_After` сравните график до и после очистки.
6. На листе `04_Pivot_Summary` покажите нативную сводную таблицу и условное форматирование.
7. На листе `05_Final_Visual` покажите формулы расчета эффекта и управленческий вывод по пилоту.
